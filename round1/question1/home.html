Website Kurukshetra


Round 1 : Participants will be given a source code of a website with few points missing and few errors.
They have to find them out and fix the website in the Frontend.
Round 2 : Participants will be given a source code of a website with few points missing and few errors.
They have to find them out and fix the website in the Backend.
Round 3 : Participants should prepare a website where the properties and specifications will be given.
There by the best website Builder will be the winner.
